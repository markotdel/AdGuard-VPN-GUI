## Установка (одна команда)
```bash
unzip adguardvpn-gui_v_._._._.zip ## Номер версии например v0.3.3.19.zip
cd adguardvpn-gui
bash install.sh
```

После установки автоматически:
- создаётся ярлык на Рабочем столе;
- добавляется пункт в меню приложений;
- открывается папка Рабочего стола, чтобы ярлык был виден сразу.

## Удаление
```bash
cd adguardvpn-gui
bash uninstall.sh
```
